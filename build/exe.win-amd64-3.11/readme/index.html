Je suis un créateur indépendant, et j'ai conçu cette application dans l'optique de permettre à chacun de sauvegarder localement n'importe quelle vidéo provenant de YouTube aux formats MP4 et MP3 sans avoir ni à payer le moindre sou ni à utiliser un navigateur Internet pour trouver un site de conversion fiable.
